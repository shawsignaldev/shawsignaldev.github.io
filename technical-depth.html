Package discipline

Python repositories use installable layouts where appropriate, small deterministic examples, and tests that do not require private credentials.

Public safety

Finance-related repositories include educational/research disclaimers and avoid sensitive account access, account data, and private records.

Recruiter usability

Every major repository provides a README, architecture documentation, security file, changelog, and role-specific review signal.
