# Ephraim Shaw Portfolio Site

Static GitHub Pages portfolio for Ephraim Shaw.

## Purpose

This site is the front door for the public GitHub portfolio. It gives recruiters and technical reviewers a fast path through the strongest repositories, package surfaces, engineering themes, and role-specific evidence.

## Local Preview

```bash
python -m http.server 8080
```

Then open `http://localhost:8080`.

## Pages

- `index.html` - executive portfolio overview
- `projects.html` - repository catalog and package map
- `review-guide.html` - role-based review path
- `technical-depth.html` - technical depth matrix

## Validation

```bash
python -m pytest -q
```
